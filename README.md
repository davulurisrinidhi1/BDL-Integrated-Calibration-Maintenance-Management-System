# Integrated Calibration & Maintenance Workflow Management System

A centralized industrial web application designed for **Bharat Dynamics Limited (BDL)** to unify and streamline **asset calibration, machine maintenance, and operational monitoring workflows** in a structured and traceable environment.

---

## 🚀 Live Deployment

🔗 https://bdl-integrated-calibration-maintenance-management-b8oqj0z7o.vercel.app

---

## 📌 Project Overview

This system provides a unified platform for managing industrial assets and maintenance operations across departments.

It integrates:

- Calibration management for instruments and gauges  
- Machine maintenance workflows (SM & PM activities)  
- Centralized dashboard monitoring  
- Asset-level traceability and lifecycle tracking  
- Due-date based maintenance visibility  

The goal is to eliminate fragmented tracking systems and enable a **single-source operational view for maintenance and calibration activities**.

---

## 🏭 Core Modules

### 1. Dashboard (Central Monitoring System)

The dashboard serves as the operational control center, providing visibility into all assets and their maintenance status.

#### 🔹 Master List (Asset Registry)

A unified repository of all industrial assets including machines, instruments, and gauges.

| Asset Code | Asset Type | Description | Department | Status | Periodicity | Last Activity | Next Due Date |

Key capabilities:
- Centralized asset tracking  
- Complete lifecycle visibility  
- Department-wise classification  

---

#### 🔹 Due List (Maintenance & Calibration Tracker)

Tracks all upcoming, pending, and overdue calibration and maintenance activities.

| Code No | Asset Type | Activity Type | Periodicity | Last Date | Due Date | Status | Next Scheduled Date | Postponement Reason |

Key capabilities:
- Real-time due monitoring  
- Status-based classification (Due / Overdue / Completed)  
- Postponement tracking with justification  
- Maintenance prioritization support  

---

### 2. Calibration Management Module

Handles calibration workflows for precision instruments and gauges such as:

- Vernier Calipers  
- Micrometers  
- Thread Plug Gauges  
- Double Ended Gauges  

Functionalities:
- Calibration entry and validation  
- GO / NO-GO decision logic  
- Certificate generation  
- Calibration history tracking  
- Traceability records for audits  

---

### 3. Machine Maintenance Module

Manages industrial machine maintenance operations such as:

- Air Compressors  
- Central AC Units  

Functionalities:
- Manual SM (Scheduled Maintenance) entries  
- Manual PM (Preventive Maintenance) records  
- Work order tracking  
- Spare part management  
- Maintenance history logging  
- Strategy-based maintenance planning

## 🧠 System Design Philosophy

The system is designed to replicate real-world industrial maintenance environments where:

- Maintenance planning is manually controlled  
- Calibration follows strict periodic compliance rules  
- Traceability and documentation are mandatory  
- Centralized monitoring improves operational decision-making  
- Audit readiness is a key requirement
